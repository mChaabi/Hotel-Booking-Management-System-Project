# 🏨 Hotel Booking Management System

A modern and intelligent Hotel Booking Management System developed using **Angular 21**, **Java 21**, **Spring Boot 3**, **PostgreSQL**, and **Spring Security**.

The platform provides a complete hotel reservation experience, allowing customers to browse available rooms, make reservations, manage bookings, and receive a seamless user experience. Administrators can efficiently manage rooms, users, reservations, and business operations through a secure dashboard.
